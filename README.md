## Примеры использования:
1. В модуле *'utils'* определена функция *'date_convert'* которая принимает строку даты в формате 'YYYY-MM-DD HH:MM:SS' и преобразовывает в datetime.
2. В модуле *'utils'* определена функция *'greeting'* которая принимает дату и время в формате 'datetime' и возвращает строку приветствия с учётом времени суток.
3. В модуле *'utils'* определена функция *'read_excel_monthly'* которая считывает операции из Excel файла и фильтрует по дате (с 1го числа месяца по заданное). Аргументы функции: путь до файла, дата в формате YYYY-MM-DD HH:MM:SS
4. В модуле *'utils'* определена функция *'cards_statistic'* которая принимает DataFrame с транзакциями и возвращает статистику по каждой карте. Возвращает список словарей с номером карты, суммой трат и кэшбэк.
5. В модуле *'utils'* определена функция *'top5_transactions'* которая принимает DataFrame с транзакциями и возвращает топ 5 транзакций по сумме. Возвращает список словарей: дата, сумма, категория, описание.
6. В модуле *'utils'* определена функция *'exchange_rates'* которая запрашивает у API данные курса валют, которые определены в JSON-файле пользователя. Необходимо добавить API - ключ в файл *'.env'* в соответствие с примером *'.env_template'*.
7. В модуле *'utils'* определена функция *'stocks_prices'* которая запрашивает у API данные цен на акции, названия которых определены во входящем списке. Необходимо добавить API - ключ в файл *'.env'* в соответствие с примером *'.env_template'*.
8. В модуле *'utils'* определена функция *'read_excel_transactions'* которая считывает операции из Excel файла и обрабатывает ошибок чтения.
9. В модуле *'utils'* определена функция *'simple_search'*. Функция поиска в столбцах Категория и Описание. На входе DF и строка поиска. На выходе отфильтрованный DF.
10. В модуле *'utils'* определена функция *'report_to_file'* Декоратор для записи результатов работы функции-отчёта в файл. Всегда сохраняет отчёты в папку data. Параметры: filename: имя файла (если None, генерируется автоматически)
11. В модуле *'views'* определена функция *'main_page'*. Функция главной страницы. Принимает на вход строку с датой в формате 'YYYY-MM-DD HH:MM:SS'. Возвращает JSON-ответ. Функция собирает в себе вспомогательные функции.
12. В модуле *'services'* определена функция *'users_search'*. Функция поиска по транзакциям (Категория, Описание). Принимает строку для поиска, возвращает JSON файл.
13. В модуле *'reports'* определена функция *'spending_by_category'*. Функция возвращает траты по заданной категории за последние три месяца (от переданной даты).
    Аргументы: transactions: дфрейм с транзакциями; category: название категории; date: опциональная дата (если не передана, берется текущая дата), если невалидный ввод, то (2021-02-12 13:00:00)
    Возвращает: JSON строку с тратами по указанной категории за последние 3 месяца.
14. В модулях настроено логирование в консоль для всех функций.
15. Excel файл с транзакциями *'operations.xlsx'*, а также отчёт генерируемый декоратором *'report_to_file'* находятся в дирректории *'data'*.
